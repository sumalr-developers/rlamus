# Rlamus

Backend of Sumlar, a mobile app to build personal knowledge bases assisted by GLMs.

## Setup

| Environmental Variable | Description             | Default Value            |
| :--------------------- | :---------------------- | :----------------------- |
| `CHROMIUM_BIN`         | Path to Chromium binary | null                     |
| `OLLAMA_ENDPOINT`      | Ollama API URL          | `http://localhost:11434` |
| `RLAMUS_MODEL`         | GLM to use              | `gemma4:12b`             |
